## Convention

- All file should save in root folder.
- Start from copying `template.html`.
- HTML file naming convention: `gradebook-word(words)-number.html`.
    - examples:
    - `gradebook-add.html`
    - `gradebook-edit-1.html`
    - `gradebook-edit-2.html`
    - `gradebook-edit-confirm-1.html`
